1. 🎯 Overview

This Nix Darwin configuration provides a complete, reproducible macOS development environment. It automatically configures system settings, installs applications, and sets up development tools with consistent theming and workflows.

Key Benefits

  • Reproducible: Same environment across multiple Macs
  • Declarative: All settings defined in code
  • Version Controlled: Track changes and rollback if needed
  • Automated: One command setup for new machines

Architecture

├── flake.nix # Main entry point and dependencies ├── hosts/darkstar/ # Host-specific configuration ├── system/ # macOS system settings ├── pkgs/ # Package definitions └── home/ # User environment (Home Manager)

3. 🚀 Installation Guide

Prerequisites

Fresh macOS installation (Apple Silicon Mac recommended)

Installation Steps

  1. Install Nix Package Manager 
    sh <(curl -L https://nixos.org/nix/install)
  2. Verify Nix Installation 
    nix-shell -p neofetch --run neofetch
  3. Clone This Repository 
    git clone <your-repo-url> ~/nix-config
cd ~/nix-config
  4. 🚨 CRITICAL: Update Personal Information

    You MUST customize the configuration before building. See the Customization section for required changes:

    • Username: "sri" → your username
    • Home directory: "/Users/stelo" → your home
    • Git info: name and email
    • AWS configuration (or remove it)
    • Host name: "darkstar" → your choice
  5. Test Configuration 
    nix flake check
  6. Initial System Build 
    nix run nix-darwin --extra-experimental-features "nix-command flakes" -- switch --flake .#YOUR-HOST-NAME
  7. Future Updates 
    darwin-rebuild switch --flake .#YOUR-HOST-NAME
⚠️ Important: The initial build may take 30-60 minutes as it downloads and compiles packages. Subsequent builds are much faster.
